---
title: Deleting an Action Type
---

![](https://usercontent.clueso.io/1f4455e0-789e-4beb-8cd9-4dac96bc47e4/95fc0fba-36b6-472a-857a-1b5a582c85b1/f163bc46-1a2e-43e3-9b51-dd57aa3fdd89/images/b0e806af-9cf7-4c56-8fb5-92e7a7ef7f7e.gif)

To delete the existing action type in Bytebeam console,

<Steps>
  <Step title="Access Settings Tab">
    Navigate to the **Settings Tab** by clicking on your username at the top right
  </Step>
  <Step title="Go to Action Types Tab and Initiate Action Type Delete">
    Navigate to the **Action Types** tab and click on the **trash** icon against the target action type
  </Step>
  <Step title="Fill Details">
    Enter the action type name and click on **Submit** button to delete the action type.
</Step>
</Steps>

<Note>The default action types cannot be deleted</Note>
