Network stats talk about the incoming and outgoing data rates of each networking interface. The "**uplink\_network\_stats**" stream has these details. This stream has the following fields:

| Field name           | Description                                                      |
| -------------------- | ---------------------------------------------------------------- |
| name                 | Name of the network interface.                                   |
| incoming\_data\_rate | The incoming data rate through the interface is in bytes/second. |
| outgoing\_data\_rate | The outgoing data rate through the interface is in bytes/second. |
