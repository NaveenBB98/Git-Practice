Processor stats talk about the frequency at which each processor is running and the usage percentage of each processor. This data is pushed to the stream "**uplink\_processor\_stats**". This stream has the below field:

| Field     | Description                           |
| --------- | ------------------------------------- |
| name      | Name of the CPU                       |
| frequency | The clock frequency of the CPU in MHz |
| usage     | Percentage of CPU used                |
