From the Bytebeam cloud, it is possible to access the shell on the Linux device. This is implemented using the amazing [tunshell](https://tunshell.com/)  project.

To shell into your device, go to the Bytebeam cloud, select the device, and under the "**Operations**" section click on the "**Remote** **Shell**" Button.

![](/assets/j18VchOxvt2qKLWyBLULC_linuxdocsshellwhite1.png)

You should now be able to see the shell environment on the cloud.

![](/assets/YRgJr9XTlnKvo4BDRStTE_linuxdocsshellwhite2.png)
