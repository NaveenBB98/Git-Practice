To add device parameters

**Step 1:** Navigate to the ***Settings Tab*** by clicking on the user name at the top right

![](/assets/5OY5oDz-qJzz4AFQfyQZ4_2023-10-2501-13.png)

**Step 2:** Add the highlighted device parameters in the device shadow stream.

![](/assets/0qcFZnhD-LakdVg12umEc_screenshot-1.jpg)



**Step 3:** You should see the device parameters in the device shadow.

![](/assets/DeaL32e_1ZN9H2Y3J1Xa3_screenshot-2.jpg)
