title: Editing a Firmware File version
---

To edit a Firmware File version in Bytebeam console,

<Steps>
  <Step title="Access Actions Tab">
    Click on the **Actions** tab
  </Step>
  <Step title="Go to Inventory Tab and Access Firmware Files Tab">
    Navigate to the **Firmware Files** tab by clicking on Inventory section
  </Step>
  <Step title="Locate the desired Firmware File version ">
    Click on **Firmware Files** tab and locate the desired **Firmware File version**
</Step>
<Step title="Click on Edit icon and Initiate Edit">
    Click on **Edit** icon against the located version, enter the new uncompressed file size along with its unit and click on **Upload** button.
</Step>
</Steps>

<Note>Version name cannot be edited and uploaded firmware file cannot be changed once uploaded.</Note>
