No Signal alert is triggered when there's an interruption in the data stream from a device for a specified duration.



To Create a No Signal Alert Rule,

**Step 1: **Navigate to the ***Alert Rules*** under the*** Alerts*** tab.

![](/assets/xzPF6etRm2yxq9Uzvlcvw_alerts-light.jpg)

**Step 2: **Click on the "*Add Alert Rule"* Button

![](/assets/pCJogDjLkRgOTAtuO7nKX_image.png)

**Step 3: **Select "*No Signal Alert"*

![](/assets/IGoYa1i5pYpW2EtG9re_W_screenshot-from-2024-03-12-22-48-14.png)

**Step 4:** Define the particular stream and the duration for which if no data is received, an alert should be triggered, specify other parameters such as criticality, threshold and click "*Submit"*

![](/assets/ZBp0ZUd6XlIiQKbBy9Ccd_screenshot-5.jpg)

You will find the new *Alert Rule* in the table on the*** Alert Rules*** tab.



![](/assets/3H2D7kpSLnt9P6_I9_74J_screenshot-4.jpg)
