To deactivate an API Key in Bytebeam Console,

<Steps>
  <Step title="Access Settings">
    Navigate to **Settings** by clicking your username at the top right.
  </Step>
  <Step title="Go to API Keys and Initiate Deletion">
    Navigate to the **API Keys** Tab and click on the delete icon against the desired API Key
  </Step>
  <Step title="Step 4">
  Type "Delete Key" in the field, and Click on **Confirm** to delete the API Key.
  </Step>

</Steps>

<Warning> As this action is irreversible, please be certain that you have chosen the right key to deactivate.</Warning>
