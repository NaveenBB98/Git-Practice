To add a Panel to the Dashboard,

**Step 1: **Go to Dashboards, click on the Dashboard

![](/assets/dmdA-fad0_xHxbS0nvw40_screenshot-4.jpeg)

In case of *the Device type dashboard,* Click on the desired Device

![](/assets/ZRXE9tqk0WqI20WCUQwlx_screenshot-5.jpeg)

**Step 2: **Click on Add Panel

![](/assets/6k1PpDJNNvFHFMEMOCuYg_screenshot-1.png)

Select from any of the given Panels.

![](/assets/UP8eGb176ydI9wU0b4JYu_screenshot-8.jpeg)

**Step 3: **Click on the desired Panel, for example, if you select the Panel Static Text, you should see

![](/assets/EoS9tnboO31PBuVeBLIzy_screenshot-10.jpeg)

**Step 5: **Enter the desired Title, Text for Static Text, and click Submit,

![](/assets/BQzrkX56NxPuwy30OVoKy_screenshot-12.jpeg)

All the Panels added to the Dashboard will be visible&#x20;

![](/assets/SkGfLLyeb6wb15lPihwb__screenshot-15.jpeg)
