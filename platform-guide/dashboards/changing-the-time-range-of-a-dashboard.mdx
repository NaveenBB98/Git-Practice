Varying intervals of time can help you visualize trends in incoming data in your Dashboard. To view a Dashboard in different time ranges,&#x20;

**Step 1:** Navigate to a Dashboard. In this example, we have selected the existing *Device Overview* Dashboard

![](/assets/h3H_TyndnlEKas4iOaJUt_screenshot-47.jpg)

**Step 2: **Select a Metadata Key to filter the Dashboard by

![](/assets/TU2OD1QfgFJIZY43LQhhm_screenshot-34.jpeg)

**Step 3: **The default time range for a Dashboard is set to *Last 5 minutes**** ***

![](/assets/Io1pj-O_Vo9BRfpzNHWbI_screenshot-35.jpeg)

**Step 4: **Changing it to *Last 1* *week* results in data from the past year

![](/assets/wY3pKrlyvsu0YDStuLLYA_screenshot-38.jpeg)
