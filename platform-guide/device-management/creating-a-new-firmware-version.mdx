To create a new Firmware Version,

**Step 1**: Navigate to the ***Firmware Versions*** tab under the*** Device Management*** tab.

![](/assets/xjA8qoxG0z-WACcADA_gN_screenshot-29.jpg)

**Step 2**: Click on the *Upload New Firmware* Button.

![](/assets/BwdqhxWEgiS5R9rHDtA7r_screenshot-30.jpg)

**Step 3**: Create New Firmware

Provide the Firmware Version Number and Click on *the Choose File* button to Upload the Firmware binary file. Then Click* *on* the Create* button to create a new Firmware Version

![](/assets/kwg7M3YrR63G14UxTsOaf_screenshot-34.jpg)

You will find the new Firmware Version in the table on the ***Firmware Versions*** tab.

![](/assets/mi9UTPRTyWrjrnIPOyyvo_screenshot-33.jpg)
