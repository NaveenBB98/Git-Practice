To create a new Configuration Version

**Step 1**: Navigate to the ***Device Configs*** tab under the*** Device Management*** tab.

![](/assets/HkakV47QjOhPbg4C8UNBu_screenshot-41.jpg)

**Step 2**: Click on the *New Device Config* Button.

![](/assets/dHRAbtsWScll4L13wjWui_screenshot-42.jpg)

**Step 3**: Create a New Device Config

Provide the Version Number,  Input the required configuration JSON, and Click* *on* the Create* button to create the new Config

![](/assets/Wd6u01qqsZhPV7Si97wxs_screenshot-43.jpg)



You will find the new *Device Config* Version in the table on the ***Device Configs*** tab.

![](/assets/xSuIuuWMQYq68T3dskY8w_screenshot-44.jpg)
