To edit Device Metadata,&#x20;

**Step 1: **Navigate to Device Management > Devices Tab

![](/assets/LprgyyMdi-koEdRLQvA6n_screenshot-28.jpg)

**Step 2:** Open the accordion of the device on which you want to edit the metadata by clicking on the device card. You will be able to view the metadata in the table.

![](/assets/XiJ2wiRZ3JaByBXEP_W_T_screenshot-31.jpg)

**Step 3: **Hovering the cursor over the Metadata value you want to edit should display a *pencil* ✎ icon

![](/assets/Tmcn7PBQD6zkucgHmz8-y_screenshot-32.jpg)

**Step 4:** Click on the *pencil *✎ icon. You should see an input box to enter the new Metadata Key value and type the desired new value. To save, click on the *checkmark* ✓ icon.&#x20;

![](/assets/MPQ_jovp8lvriWgYa31Au_screenshot-35.jpg)



You should see the new metadata value in the table now

![](/assets/abFykx-62h95WnWU08fix_screenshot-36.jpg)
