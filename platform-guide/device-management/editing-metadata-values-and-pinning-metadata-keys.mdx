You can update values of metadata key for any device, making it easier to identify devices when using filters.

To edit metadata key values in the Bytebeam console, follow these steps:

<Steps> 
<Step title="Access Device Management Tab"> Click on the **Device Management** tab at the top. </Step> 
<Step title="Locate the Device"> Use pagination to find the device if there are many devices available. 
<Note>You can use metadata keys or Device ID as filters to locate the desired device more easily.</Note> </Step> 
<Step title="Edit Metadata Key Value"> 
1. Open the Device accordion. 
2. Hover over the existing value of the desired metadata key. 
3. Click on the edit icon. 
4. Enter a new value or replace the existing one. 
5. Click the check icon to save the update. 
</Step> 
</Steps>
<Note>To improve visibility in the Devices table, you can pin or unpin metadata keys by clicking on the Pin icon when you hover over the metadata key.</Note>
