To create a new metadata key in Bytebeam console,

<Steps>
  <Step title="Access Settings Tab">
    Navigate to the **Settings Tab** by clicking on your username at the top right
  </Step>
  <Step title="Go to Metadata Tab and Initiate Metadata Key Creation">
    Navigate to the **Metadata** tab and click on the **Create Metadata Key** button
  </Step>
  <Step title="Fill Details">
    Type the metadata key name and click on **Submit** button to create the metadata key
</Step>
</Steps>
