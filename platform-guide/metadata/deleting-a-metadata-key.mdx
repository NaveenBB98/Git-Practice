---
title: Deleting a Metadata Key
---

![](https://usercontent.clueso.io/1f4455e0-789e-4beb-8cd9-4dac96bc47e4/0359be30-7b32-4932-a5d0-d6fc90ecd8bf/a0af0905-f11e-41b5-b52e-98ec3f4f01a3/images/13b94a3d-4ab2-40fe-b73d-46d0c780e3f5.gif)

To delete the existing metadata key in Bytebeam console,

<Steps>
  <Step title="Access Settings Tab">
    Navigate to the **Settings Tab** by clicking on your username at the top right
  </Step>
  <Step title="Go to Metadata Tab and Initiate Metadata Key Deletion">
    Navigate to the **Metadata** tab and click on the **trash**(ğŸ—‘) icon against the target metadata key
  </Step>
  <Step title="Fill Details">
    Enter the metadata key name and click on **Submit** button to delete the metadata key
</Step>
</Steps>

<Note>When a metadata key is set as a serial key, it cannot be deleted until the serial key is unset.</Note>
