title: Editing a Metadata Key
---

To edit the existing metadata key in Bytebeam console,

<Steps>
  <Step title="Access Settings Tab">
    Navigate to the **Settings Tab** by clicking on your username at the top right
  </Step>
  <Step title="Go to Metadata Tab and Initiate Metadata Key Edit">
    Navigate to the **Metadata** tab and click on the **edit** icon against the target metadata key
  </Step>
  <Step title="Fill Details">
    Enter the changed metadata key name and click on **Submit** button to edit the metadata key
</Step>
</Steps>

<Note>When a metadata key is set as a serial key, it cannot be edited until the serial key is unset.</Note>
