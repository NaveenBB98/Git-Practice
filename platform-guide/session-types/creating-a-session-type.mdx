To create a new session type in Bytebeam console,

<Steps>
  <Step title="Access Settings Tab">
    Navigate to the **Settings Tab** by clicking on your username at the top right
  </Step>
  <Step title="Go to Session Types Tab and Initiate Session Type Creation">
    Navigate to the **Session Types** tab and click on the **+ Create Session Type** button
  </Step>
  <Step title="Fill Details">
    Type the session type name, add the conditions by selecting the stream and fields and click on **Submit** button to create the session type.
</Step>
</Steps>

<Note>Use the icons next to each created session type to start or stop the session.</Note>
