title: Deleting a Session Type
---

![](https://usercontent.clueso.io/1f4455e0-789e-4beb-8cd9-4dac96bc47e4/a9501e10-33d1-4c78-a572-ef1d5a0ae159/e6a8f1ae-babd-4a51-93e4-c400dec6e62a/images/53fde410-d382-4661-97eb-aea6015a7a75.gif)

To delete the existing session type in Bytebeam console,

<Steps>
  <Step title="Access Settings Tab">
    Navigate to the **Settings Tab** by clicking on your username at the top right
  </Step>
  <Step title="Go to Session Types Tab and Initiate Session Type Deletion">
    Navigate to the **Session Types** tab and click on the **trash**(ðŸ—‘) icon against the target session type
  </Step>
  <Step title="Fill Details">
    Enter the session type name and click on **Submit** button to delete it.
</Step>
</Steps>

<Warning>When a session type is running, it cannot be deleted until it is stopped.</Warning>
