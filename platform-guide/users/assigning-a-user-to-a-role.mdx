To edit an existing user,

<Steps>
  <Step title="Access Settings Tab">
    Navigate to the **Settings Tab** by clicking on your username at the top right
  </Step>
  <Step title="Go to User Tab and Initiate User Edit">
    Navigate to the **Users** tab, locate the user click on the **edit** icon against it.
  </Step>
  <Step title="Fill Details">
    Change the details like email, select a different role from dropdown and click on **Submit** button to edit the user.
</Step>
</Steps>
