To delete a User in Bytebeam console,

<Steps>
<Step title="Access Settings Tab">
    Navigate to the **Settings** tab by clicking on your username at the top right
  </Step>
  <Step title="Locate the Target User">
    Go to **Roles** tab and scroll to the **User** which needs to be deleted
  </Step>
  <Step title="Initiate the User Deletion">
    Click on the **Trash** ðŸ—‘ icon against the user you would like to delete
  </Step>
  <Step title="Confirm the selection">
    Type user name in the field, and Click on **Confirm** to delete it.
  </Step>
</Steps>

<Note>As this action is irreversible, please be certain that you have chosen the right User to delete.</Note>
